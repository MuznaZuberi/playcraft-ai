🚀 **PlayCraft AI** is a modern, interactive web-based project built to showcase creative UI design, smooth animations, and engaging user experiences using core web technologies.

The project focuses on delivering a **clean, responsive, and visually appealing interface** while maintaining simple and efficient front-end architecture.

##### Interactive elements and smooth transitions for better user experience.

##### 📱 **Responsive Layout**
##### Fully responsive design that adapts to different screen sizes.
